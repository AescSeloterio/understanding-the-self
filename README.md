# 🪞 Unpacking the Self

A modern, interactive educational web guide designed to help users explore and reflect on the six core dimensions of identity.

---

## 📖 Overview

**Unpacking the Self** provides a structured, guided journey through key psychological and philosophical concepts framing human identity. Built with a responsive layout, persistent progress tracking, and interactive study tools, it serves as an accessible learning module for students and self-explorers alike.

### Key Dimensions Covered
- 🏋️ **Physical Self** — Body image, embodiment, and physical presence.
- 💭 **Emotional Self** — Emotional intelligence, affect labeling, and regulation.
- 🧠 **Intellectual Self** — Growth mindset, curiosity, and lifelong learning.
- 👥 **Social Self** — Looking-glass self, social roles, and belonging.
- ✨ **Spiritual Self** — Purpose, inner values, and search for meaning.
- 🌐 **Digital Self** — Online personas, digital footprints, and balance.

---

## ✨ Features

- **Interactive Lessons:** Organized module cards with expandable subtopics.
- **Progress Tracking:** LocalStorage integration to automatically save your reading progress.
- **Clean UI:** Styled with Tailwind CSS for an elegant, distraction-free reading experience.
- **Dynamic Content:** Lightweight, modular setup using vanilla JavaScript.
